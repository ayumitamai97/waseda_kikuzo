# kikuzo.rb の使い方
## コマンドの実行
学外の場合のみ、コマンドラインの第一引数にMyWasedaのID, 第二引数にMyWasedaのパスワードを入力する。

## 学内、学外の切り替え
configのコメントアウト部分とメソッドの呼び出しのコメントアウト部分を適宜入れ替えるなどする。

## 検索条件の変更
適宜`conditions`の中身や年月日を書き換える。
